Embedding the Spokane Shelter Map

Instructions for GrowthZone CMS — Spokane Business Association

Before you start: Replace https://YOUR-DOMAIN throughout these examples with the actual deployed URL of the shelter dashboard (e.g. https://shelter.spokanebusinessassociation.com).

Method 1 — Recommended iFrame Embed

The standard approach. GrowthZone supports embedding external pages via its Custom HTML or Embed Code widget.

Steps

  1. Log in to your GrowthZone Admin panel.
  2. Go to Website → Pages and open the page you want to edit.
  3. Click Add Widget (or the + icon in the page builder).
  4. Choose Custom HTML or Embed Code.
  5. Paste the code below and click Save / Publish.
<iframe
  src="https://YOUR-DOMAIN/embed"
  width="100%"
  height="620"
  style="border: none; border-radius: 8px; display: block;"
  title="Spokane Shelter Availability Map"
  loading="lazy"
  allow="geolocation">
</iframe>

Optional URL parameters

ParameterExampleEffect
filter /embed?filter=families Pre-selects a filter pill. Valid values: all, men, women, youth, young-adults, families, lgbtq, dv, low-barrier, high-barrier

Example — open with Families filter pre-selected:

<iframe
  src="https://YOUR-DOMAIN/embed?filter=families"
  width="100%"
  height="620"
  style="border: none; border-radius: 8px; display: block;"
  title="Spokane Shelter Availability Map — Families"
  loading="lazy">
</iframe>

Method 2 — Fallback JavaScript Embed

Some GrowthZone setups sanitize raw <iframe> tags in the HTML editor. Use this JavaScript snippet if Method 1 doesn’t render — it builds the iframe dynamically after the page loads, bypassing most editor restrictions.

<div id="spokane-shelter-map"
     style="width:100%;height:620px;border-radius:8px;overflow:hidden;"></div>
<script>
(function () {
  var el = document.getElementById('spokane-shelter-map');
  if (!el) return;
  var f = document.createElement('iframe');
  f.src   = 'https://YOUR-DOMAIN/embed';
  f.style.cssText = 'width:100%;height:100%;border:none;display:block;';
  f.title = 'Spokane Shelter Availability Map';
  f.setAttribute('loading', 'lazy');
  el.appendChild(f);
})();
</script>

Troubleshooting

  • Blank gray area — Check that the hosting server has outbound HTTPS access (port 443). The map tiles load from CartoDB’s CDN and require internet access.
  • “Refused to display in a frame” — The dashboard sets Content-Security-Policy: frame-ancestors * on all responses, so this should not happen. If it does, check whether a CDN or reverse-proxy is stripping or overwriting response headers.
  • Map is cut off / scrollbar appears — Increase the height on the <iframe> element. 620px fits comfortably on most desktop layouts. Try 500px for narrow columns.
  • Filter pills not visible on mobile — The slim header bar is horizontally scrollable. Swipe left inside the header to reveal additional pills.
  • GrowthZone “Custom HTML” not available — Contact your GrowthZone account manager; the Custom HTML widget is available in most plans under Website → Site Builder → Widget Library.
